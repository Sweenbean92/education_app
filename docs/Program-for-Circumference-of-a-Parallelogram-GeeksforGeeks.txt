Given the sides of Parallelogram then calculate the circumference.Examples :

Input: a = 10, b = 8 Output: 36.00 Input: a = 25.12, b = 20.4 Output: 91.04

The opposite sides of a parallelogram are of equal length and parallel. The angles are pairwise equal but not necessarily 90 degree. The circumference of a Parallelogram can be computed as sum of two adjacent sides each multiplied by 2.

Formula used to calculate circumference of a Parallelogram: (2*a)+(2*b)

// C Program to calculate the // Circumference of a Parallelogram. #include <stdio.h> float circumferenceparallelogram(float a, float b) { return ((2 * a) + (2 * b)); } int main() { float a = 10, b = 8; printf("Circumference of a given Parallelogram is : %.2f", circumferenceparallelogram(a, b)); return 0; }
